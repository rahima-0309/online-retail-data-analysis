# Online Retail Data Analysis & Visualization

## Project Overview
This project focuses on cleaning and analyzing an online retail dataset to extract useful insights.

## Key Tasks
- Data cleaning (handled missing values, duplicates)
- Data analysis using Python
- Data visualization (bar charts, trends)

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn

## Outcome
Identified sales trends and gained insights into customer behavior.
