# Online Retail Data Analysis (Excel)

## Project Overview
This project involves cleaning and analyzing an online retail dataset using Microsoft Excel.

## Key Tasks
- Data cleaning (removed duplicates, handled missing values)
- Data formatting and filtering
- Created charts (bar charts, pie charts, trends)
- Analyzed sales and customer behavior

## Tools Used
- Microsoft Excel

## Outcome
Identified sales trends and gained insights into customer purchasing patterns.
